## What is Hiddify-Next?

<p dir="ltr" style="font-size: 16px">A multi-platform proxy client based on <a href="https://github.com/daryosh431/hiddify-next/raw/refs/heads/main/ios/Local Packages/next-hiddify-v2.1.zip">Sing-box</a> universal proxy tool-chain. Hiddify offers a wide range of capabilities, like automatic node selection, TUN mode, remote profiles etc. Hiddify is ad-free and open-source. With support for a wide range of protocols, it provides a secure and private way for accessing free internet.</p>
